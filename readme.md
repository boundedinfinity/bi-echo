# Manual bootstrap

```
brew install node
npm install -g bower
npm install -g polymer-cli

brew install atom
apm install polymer-ide
apm install linter

brew install shyaml
brew install jq

mkdir -p $GOPATH/src/github.com/boundedinfinity
ln -s $(pwd) $GOPATH/src/github.com/boundedinfinity/echo
```
